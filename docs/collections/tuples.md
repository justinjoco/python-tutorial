## why to use tuples
## Creating a tuple
```python
empty_tuple = ()
packed_tuple = (2, 5 , 7, 3, 2, 1, 9)
```
## Accessing a tuple element
```python
print(packed_tuple[1])

print(packed_tuple[1:5])
```
## Unpacking a tuple
```python
date_tuple = ("04", "05", "2019")

(month, day, year) = date_tuple
print(month)
print(day)
print(year)
```
